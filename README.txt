How to publish this site on GitHub Pages

1) Create a new public repository on GitHub.
2) Upload all files and folders from this package exactly as they are.
3) Make sure index.html is in the root of the repository.
4) Go to Repository Settings > Pages.
5) Under Build and deployment, choose:
   Source: Deploy from a branch
   Branch: main (or master) / root
6) Save.
7) After a short wait, GitHub will publish the site.

Notes:
- The German CV file is already included.
- The English CV button is currently a placeholder because no English PDF was uploaded yet.
- You can replace text, LinkedIn link, email, phone number, or images later.
